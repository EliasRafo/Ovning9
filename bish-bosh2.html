Bish-Bosh lists all numbers between 1 and any number entered by user. If the number is evenly divisible by first division number, 'Bish' is listed instead of the number. If the number is evenly divisible by second division number, 'Bosh' is listed instead of the number. If the number is evenly divisible by both first and second division numbers, "Bish-Bosh" is written instead of the number.
